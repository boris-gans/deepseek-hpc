We use Meta's 7B parameter model from their OpenLlama series. You can download it from: https://huggingface.co/openlm-research/open_llama_7b , or via the HuggingFace CLI with 

```bash
hf download openlm-research/open_llama_7b --include="*" --local-dir ~/[path-to-project]/models
```

*Note: you must authenticate your device with HF first*
